# 🎵 Optical Music Box

A Flutter mobile app for kids that reads hand-drawn black dots on a 5-string paper strip and plays pentatonic notes in real-time.

---

## How it works

1. **Print or draw** a paper strip with 5 rows (matching the 5 coloured zones).
2. **Hold the strip** in front of your phone camera, centred in the glowing playhead.
3. **Slide the strip** slowly. When a black dot crosses a zone, that note plays instantly.

### Note mapping (top → bottom)

| Zone | Colour | Note |
|------|--------|------|
| 0    | 🔴 Red    | A4 (440 Hz) |
| 1    | 🟠 Orange | G4 (392 Hz) |
| 2    | 🟡 Yellow | E4 (330 Hz) |
| 3    | 🟢 Green  | D4 (294 Hz) |
| 4    | 🔵 Blue   | C4 (262 Hz) |

## Architecture

```
lib/
├── main.dart                    # Entry point, camera init
├── screens/
│   └── music_box_screen.dart   # Camera lifecycle, image stream, UI
├── engine/
│   ├── dot_detector.dart       # YUV420 luminance sampling (5 zones)
│   └── note_trigger.dart       # State-toggle: rising-edge audio trigger
├── audio/
│   └── audio_manager.dart      # soundpool loader + playNote(i)
└── ui/
    └── playhead_painter.dart   # CustomPainter overlay + colour/name constants
assets/sounds/                  # 5 pentatonic WAV files (generated by tools/gen_sounds.py)
```

### Detection Engine

- Reads the **Y-plane** (luminance) of each `CameraImage` frame.
- For each of the 5 zones, samples a **24×24 pixel region** centred on the playhead.
- If mean luminance **< 85** → dot detected → play note (once per crossing).
- Frame rate is throttled to ~**30 fps** for analysis.

## Getting started

```bash
# Generate audio assets (only needed once)
python3 tools/gen_sounds.py

# Install dependencies
flutter pub get

# Run on device
flutter run
```

> **Android** requires a physical device (camera emulation is limited).  
> **iOS** requires a signed provisioning profile.

---

## Tuning

Open `lib/engine/dot_detector.dart` and adjust:

- `_luminanceThreshold` (default `85`) — lower = needs darker dots; raise for low-light environments.
- `_sampleRegionPx` (default `24`) — size of the sampled square in pixels.
